# 火场音频回传 PC端

> 作者:一棵桔子 | 版本:V2.3

## 功能特点

- 🎚️ 实时音量显示
- 🔊 静音检测自动分段
- 📱 多设备支持
- 🎵 自动转MP3
- 📝 实时日志

## 环境要求

- Python 3.7+
- pygame
- ffmpeg (可选，用于转MP3)

## 安装

```bash
pip install pygame
```

## 使用方法

1. 运行 `python receiver_gui_v2.py`
2. 设置端口(默认8080)和静音阈值
3. 点击"启动服务"
4. 安卓端连接后自动接收录音

## 配置

- 端口: 8080
- 静音阈值: 500 (低于此值不录音)
- 静音分段: 3秒

## 文件保存

录音文件保存在 `received/` 目录下，按设备ID分类。
